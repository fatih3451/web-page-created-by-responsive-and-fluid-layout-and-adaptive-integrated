# Java Starter Project

A simple Java command-line starter project.

## Files

- `src/Main.java` - main application entry point.

## Run locally

1. Open a terminal.
2. Change directory to this project:
   ```bash
   cd ~/java-starter-app
   ```
3. Compile the code:
   ```bash
   javac -d out src/Main.java
   ```
4. Run the program:
   ```bash
   java -cp out Main
   ```

## Next steps

- Edit `src/Main.java` to print new messages.
- Add new classes in `src/`.
- If you have a Java IDE installed, open this folder and run `Main` directly.
